<AnalyticsArguments />

## Response

<ResponseField name="url" type="string" required>
  The destination URL
</ResponseField>

<ResponseField name="clicks" type="number" required>
  The number of clicks from this URL
</ResponseField>

<RequestExample>

```typescript
await dub.analytics.topUrls({
  key: "dummy",
  domain: "dub.sh",
  interval: "30d",
});
```

</RequestExample>

<ResponseExample>

```typescript
[
  { url: "https://dub.co/", clicks: 13 },
  { url: "https://dub.co/?qr=", clicks: 4 },
];
```

</ResponseExample>
