<AnalyticsArguments />

## Response

<ResponseField name="referer" type="string" required>
  The name of the referer. If unknown, this will be `(direct)`
</ResponseField>

<ResponseField name="clicks" type="number" required>
  The number of clicks from this referer
</ResponseField>

<RequestExample>

```typescript
await dub.analytics.referers({
  key: "dummy",
  domain: "dub.sh",
  interval: "30d",
});
```

</RequestExample>

<ResponseExample>

```typescript
[
  { referer: "(direct)", clicks: 14 },
  { referer: "t.co", clicks: 12 },
];
```

</ResponseExample>
