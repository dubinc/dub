  higher.
</Note>

<AnalyticsArguments />

## Response

<ResponseField name="os" type="string" required>
  The name of the OS
</ResponseField>

<ResponseField name="clicks" type="number" required>
  The number of clicks from this OS
</ResponseField>

<RequestExample>

```typescript
await dub.analytics.os({
  key: "dummy",
  domain: "dub.sh",
  interval: "30d",
});
```

</RequestExample>

<ResponseExample>

```typescript
[
  { os: "Mac OS", clicks: 19 },
  { os: "Windows", clicks: 12 },
];
```

</ResponseExample>
