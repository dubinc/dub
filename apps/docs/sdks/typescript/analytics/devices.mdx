<AnalyticsArguments />

## Response

<ResponseField name="device" type="string" required>
  The name of the device
</ResponseField>

<ResponseField name="clicks" type="number" required>
  The number of clicks from this device
</ResponseField>

<RequestExample>

```typescript
await dub.analytics.devices({
  key: "dummy",
  domain: "dub.sh",
  interval: "30d",
});
```

</RequestExample>

<ResponseExample>

```typescript
[
  { device: "Desktop", clicks: 15 },
  { device: "Mobile", clicks: 12 },
];
```

</ResponseExample>
